- [Discover pools](https://docs.lpagent.io/api-reference/pools/discover-pools.md): Retrieve a list of pools with filtering, sorting, and pagination
- [Get detailed information for a specific pool](https://docs.lpagent.io/api-reference/pools/get-detailed-information-for-a-specific-pool.md): Retrieve token info, liquidity visualization, and pool data for a specific pool
- [Get onchain statistics for a specific pool](https://docs.lpagent.io/api-reference/pools/get-onchain-statistics-for-a-specific-pool.md): Retrieve aggregated statistics including total open positions, input values, and unique users for a pool
- [Get positions for a specific pool](https://docs.lpagent.io/api-reference/pools/get-positions-for-a-specific-pool.md): Retrieve all positions associated with a specific pool along with position state and pricing info
- [Get top LPers for a pool](https://docs.lpagent.io/api-reference/pools/get-top-lpers-for-a-pool.md): Retrieve ranked LP providers for a pool with pagination and sorting
- [Zap-In — Generate transaction to add liquidity to a pool](https://docs.lpagent.io/api-reference/pools/zap-in-—-generate-transaction-to-add-liquidity-to-a-pool.md): Generate serialized zap-in transactions to add liquidity to a pool. Supports both DLMM and DAMM V2 pools (auto-detected by pool address). In zap-in mode, provide SOL and the API auto-swaps to the correct token ratio. Returns serialized transactions ready to be signed and landed.
- [Zap-In — Submit and land zap-in transactions](https://docs.lpagent.io/api-reference/pools/zap-in-—-submit-and-land-zap-in-transactions.md): Submit signed zap-in transactions via Jito bundles for on-chain execution
- [Zap-Out — Generate transaction to withdraw liquidity](https://docs.lpagent.io/api-reference/position/zap-out-—-generate-transaction-to-withdraw-liquidity.md): Generates serialized zap-out transactions to withdraw liquidity from a position. Supports both Meteora DLMM and DAMM V2 positions.
- [Zap-Out — Get quotes for withdrawing liquidity](https://docs.lpagent.io/api-reference/position/zap-out-—-get-quotes-for-withdrawing-liquidity.md): Returns quotes for converting position tokens to different output combinations (token0 to token1, token1 to token0, to base token, etc.) before executing a zap-out
- [Zap-Out — Submit and land zap-out transactions](https://docs.lpagent.io/api-reference/position/zap-out-—-submit-and-land-zap-out-transactions.md): Submit signed zap-out (decrease position) transactions via Jito bundles for on-chain execution. Provides better landing rate than direct RPC submission.
- [Get historical LP positions for an owner](https://docs.lpagent.io/api-reference/positions/get-historical-lp-positions-for-an-owner.md): Retrieves historical liquidity pool positions for a specific wallet owner
- [Get logs for LP positions](https://docs.lpagent.io/api-reference/positions/get-logs-for-lp-positions.md): Retrieves transaction logs for liquidity pool positions. Can fetch logs for a specific position or for all positions of an owner.
- [Get LP positions revenue for an owner](https://docs.lpagent.io/api-reference/positions/get-lp-positions-revenue-for-an-owner.md): Retrieve revenue and PnL data over time for a specific wallet owner. Only 7D and 1M ranges are available.
- [Get opening LP positions for an owner](https://docs.lpagent.io/api-reference/positions/get-opening-lp-positions-for-an-owner.md): Retrieves current liquidity pool positions that are open for a specific wallet owner
- [Get overview metrics for LP positions](https://docs.lpagent.io/api-reference/positions/get-overview-metrics-for-lp-positions.md): Retrieves comprehensive overview and metrics for liquidity pool positions of a specific owner
- [Get token balances for a wallet address](https://docs.lpagent.io/api-reference/token/get-token-balances-for-a-wallet-address.md): Returns all token balances (SOL and SPL tokens) for the specified wallet address
